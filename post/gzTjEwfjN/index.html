重读深入理解Java虚拟机（三）—— 读深入理解JVM之类文件结构

2021-04-12 8 min read # JVM # 读书笔记

类文件结构

最近在读 深入理解 Java虚拟机，这里列出类文件结构部分大纲，还需配合该书进行学习。

无关性

  • 字节码存储格式
    • 虚拟机只与字节码文件进行绑定
    • 安全性：对字节码文件有很多强制性约束
  • 虚拟机
    • 虚拟机只执行字节码文件，而不关心它的来源
    • 字节码文件描述能力比Java语言更强大

Class类文件的结构

注意

  • 每一个Class文件都对应唯一的类或接口，但这个类或接口不一定在文件中，可以通过类加载器直接生成
  • 以8字节为基础单位的二进制流
    • 占用8位以上空间的数据项需要分割
      • 分割方式：高位在前
  • 使用伪结构来存储数据

两种数据类型

  • 无符号数
    • 基本数据类型
    • 分类
      • u1
      • u2
      • u4
      • u8
  • 表
    • 复合数据类型
      • 有层次
    • 由多个无符号数组成

集合

  • 有同一类型但数量不定的多个数据时
    • 前置容量计数器

文件格式

  • magic
    • 魔数：CAFE BABE
    • 标识文件类型
    • u4
  • 版本
    • minor_version
      • Class文件的次版本号
      • u2
    • major_version
      • Class文件主版本号
      • u2
  • 常量集合
    • constant_pool_count
      • 常量池中常量个数计数
      • u2
    • constant_pool
      • 常量池
      • 长度为constant_pool_count - 1
        • 第0号常量池表示指向常量池的索引值数据表达不引用任何一个常量池项目
      • 常量池的两大类常量
        • 字面量
          • 字符串，声明为final的常量值
          • 运行期不发生动态链接
        • 符号引用
          • 包含的三类常量
            • 类和接口的全限定名
            • 字段的名称和描述符
            • 方法的名称和描述符
          • 符号引用在运行期间经过转换得到真正的内存入口地址
      • 常量池中的常量
        • 每个常量都是表结构
          • 表结构第一项都是tag，表示常量的类型
        • 14中常量有自己的表结构
          • 即有不同的表示
        • 在字段表、方法表、属性表中存在对这些常量的引用
  • access_flags
    • 访问标志
    • 标识
      • 是否public
      • 是否final
      • 是否允许使用invokespecial字节码指令的新语意
        • JDK 1.0.2后都为真
        • 改指令语意发生过改变
      • 是否是接口
      • 是否是abstract
      • 标识该类不是由用户代码产生
      • 标识是注解
      • 标识是枚举
    • u2
  • this_class
    • 类索引
    • u2
  • super_class
    • 父类索引
    • u2
  • 接口索引集合
    • interfaces_count
      • 接口索引计数
      • u2
    • interfaces
      • 接口索引集合
  • 字段表集合
    • fields_count
      • 字段表集合中字段的个数
      • u2
    • fields
      • 字段表集合
        • 字段是接口或类中声明的变量
        • 在Class文件中字段以字段表的形式存储
        • 除属性表集合外其他均为u2类型
      • 字段包含的信息
        • 作用域
        • 实例变量还是类变量
        • 可变性
        • 并发可见性
        • 可否序列化
        • 字段数据类型
        • 字段名称
      • 字段表结构
        • access_flag
          • 访问类型
        • name_index
          • 字段名称
          • 指向常量池
        • descrptor_index
          • 方法描述
          • 指向常量池
        • attribute_count
        • attributes
      • 描述符规则
        • 基本数据类型使用大写字母表示
        • 对象类型使用L加对象全限定名
  • 方法表集合
    • methods_count
      • 方法集合的计数
      • u2
    • methods
      • 方法表集合
        • 方法在Class文件中以方法表的形式存储
      • 方法表结构
        • access_flag
        • name_index
          • 指向常量池
          • 常量池中存方法名
        • descriptor_index
          • (方法参数)返回值类型
        • attribute_count
        • attributes
          • Code属性存放了方法体字节码指令
      • 注意
        • 若没有重写，不会出现父类方法信息
  • 属性表集合
    • attributes_count
    • attributes
      • 属性表结构
        • attribute_name_index
          • u2
          • 指向常量池来获取名称
        • attribute_length
          • u4
          • 使用该字段指定长度
          • 以字节为单位
        • info
          • u1
          • 数量为attribute_length定义的数量
          • 属性结构完全自定义
      • 预定义的属性
        • Code
          • Code属性表的结构
            • attribute_name_index
            • attribute_length
            • max_stack
              • 操作数栈最大深度
              • 用来分配栈帧中操作数栈深度
            • max_locals
              • 局部变量表所需的存储空间
              • 单位slot
              • 并不是所有变量所占slot的和
                • 注意slot可以重用，当代码执行超过该局部变量范围时，slot可以重新赋值
            • code_length
              • Java源程序编译后的字节码指令长度
              • u4类型
              • 但只能使用u2的长度
            • code
              • 存储字节码指令一系列字节流
                • 每个指令都是u1的单字节
            • 异常表集合
              • 不是必须存在
              • 异常表结构
                • start_pc
                  • 开始行
                    • 字节码相对于方法体的偏移量
                • end_pc
                  • 结束行
                • handler_pc
                  • 出现异常跳转行
                • catch_type
                  • 异常类型
                    • 指向一个CONSTANT_Class_info型常量的索引
                    • 当该值为0时，表示任意异常都要转到handler_pc进行处理
              • 使用异常表而不是跳转来完成异常处理
        • Exceptions属性
          • 列举方法中可能抛出的受检查的异常
            • throws关键字后的异常
          • 属性表结构
            • attribute_name_index
            • attribute_length
            • number_of_exceptions
              • 抛出受检查异常的种类数
            • exception_index_table
              • 指向常量池中CONSTANT_Class_info型常量的索引
                • 代表受查异常的类型
        • LineNumberTable属性
          • Java源码行号与字节码行号的对应关系
            • 行号：偏移量
          • 属性结构
            • attribute_name_index
              • 属性名
            • attribute_length
              • 属性长度
            • line_number_table_length
              • 映射表长度
            • line_number_table
              • 集合
                • 大小：line_number_table_length
                • 类型：line_number_info
                  • line_number_info包括start_pc和line_number两个u2类型的数据项
        • LocalVariableTable属性
          • 栈帧中局部变量表中的变量与Java源码中定义的变量之间的关系
            • 缺失时IDE会使用args0等占位符来代替
          • 属性结构
            • attribute_name_index
              • 属性名
              • 指向常量池
            • attribute_length
              • 属性长度
            • local_variable_table_length
              • 映射表表长
            • local_variable_table
              • 大小：local_variable_table_length指定
              • 类型：local_variable_info
                • 结构
                  • start_pc
                    • 局部变量生命周期开始的字节码偏移量
                  • length
                    • 局部变量作用范围覆盖的长度
                  • name_index
                    • 局部变量的名称
                    • 指向常量池
                  • descriptor_index
                    • 局部变量的描述符
                    • 指向常量池
                  • index
                    • 局部变量在栈帧局部变量表中Slot的位置
                    • 若为64位类型时，占用index和index+1两个位置
                • 与LocalVariableTypeTable的区分
                  • 描述泛型
                  • 描述符中泛型的参数化类型被擦除，描述符不能准确描述泛型类型
                  • 将descriptor_index换为字段的特征签名Signature
        • SourceFile属性
          • 记录生成Class文件的源码文件的名称
          • 可选属性
          • 结构
            • attribute_name_index
            • attribute_length
            • sourcefile_index
              • 指向常量池
              • 源码文件名
        • ConstantValue属性
          • 通知虚拟机为静态变量赋值
          • 变量的赋值方式
            • 非静态变量
              • 实例构造器方法进行
            • 静态变量
              • 类构造器中
              • 使用Constant属性
                • 使用final来修饰，并且数据类型是基本数据类型或String类型
          • 结构
            • attribute_name_index
            • attribute_length
              • 定长属性，值为2
            • constantvalue_index
              • 常量池中一个字面量的引用
              • u2数据类型
        • InnerClasses属性
          • 记录内部类与宿主类之间的关联
          • 对外部类和它的内部类生成该属性
          • 结构
            • attribute_name_index
            • attribute_length
            • number_of_classes
              • 表示要记录多少个内部类信息
            • inner_classes
              • 类型：inner_class_info
                • inner_class_info_index
                  • 指向常量池
                  • 内部类的符号引用：CONSTANT_Class_info型常量
                • outer_class_info_index
                  • 指向常量池
                  • 外部类的符号引用
                • inner_name_index
                  • 指向常量池
                  • 内部类的名字
                    • 匿名类值为0
                    • 即常量池的0号
                • inner_class_access_flags
                  • 内部类的访问标志
              • 大小：number_of_classes指定
        • Deprecated属性
          • 过时标志
          • 有或没有，没有属性值
          • 结构
            • attribute_name_index
            • attribute_length
        • Synthetic属性
          • 表示该类、字段或方法是由编译期自动产生的
        • StackMapTable
          • 位于Code属性表中的变长属性
          • 虚拟机类加载的字节码验证阶段被新类型检查验证器使用
          • 在编译期间将一系列验证信息直接记录在Class文件中
          • 结构
            • attribute_name_index
            • attribute_length
            • number_of_entries
            • stack_map_frame entries
              • 大小：number_of_entries指定
              • 类型：stack_map_frame
                • 显式或隐式代表一个字节码偏移量
                • 表示执行该字节码时局部变量表和操作数栈的验证类型
        • Signature属性
          • 可选的定长属性
          • 记录泛型中指定的类型变量或参数化类型
          • 由于Java中泛型时伪泛型，使用这个属性可以使用反射获取泛型类型
          • 结构
            • attribute_name_index
            • attribute_length
            • signature_index
              • 指向常量池中的签名
        • BootstrapMethods属性
          • 变长属性，位于类文件属性表中
          • 保存invokedynamic指令引用的引导方法限定符
          • 类文件结构的常量池中出现CONSTANT_InvokeDynamic_info类型的常量
          • 结构
            • attribute_name_index
            • attribute_length
            • num_bootstrap_methods
              • 引导方法限定符的数量
            • bootstrap_methods
              • 大小：num_bootstrap_methods指定
              • 类型：bootstrap_method
                • bootstrap_method_ref
                  • 对常量池中CONSTANT_MethodHandle_info结构的指向
                • num_bootsrap_arguments
                  • 指出静态参数个数
                • bootstrap_arguments
                  • 大小：num_bootsrap_argumnets指出
                  • 类型：对常量池的有效索引
